# Weather India

A simple public weather site with an Indian-inspired design, built with Open-Meteo.

## Files

- `index.html` – main page
- `styles.css` – styling
- `script.js` – weather logic

## Usage

1. Open `index.html` in your browser.
2. Enter a city name like `Delhi`, `Mumbai`, or `Bangalore`.
3. Click **Search** or press Enter.

This app uses Open-Meteo's geocoding and forecast endpoints, so no API key is required.

## Make it public

To open this site from anywhere on a phone or desktop, publish it to a static hosting service and use the generated public link.

### Recommended options

#### GitHub Pages
1. Create a GitHub repository and commit these files.
2. Push the repository to GitHub.
3. Enable GitHub Pages from the `main` branch or `gh-pages` branch.
4. Your public URL will look like:
   - `https://<your-username>.github.io/<repo-name>/`

#### Netlify
1. Sign in to Netlify.
2. Drag this project folder onto the Deploy page, or connect your GitHub repo.
3. Netlify assigns a public URL automatically.

#### Vercel
1. Sign in to Vercel.
2. Import the repository.
3. Deploy the site and get a public link.

## Direct viewing on phone

- If you want a true public link, publish to GitHub Pages, Netlify, or Vercel.
- After deployment, the service will give you a URL that you can open on any phone browser.
